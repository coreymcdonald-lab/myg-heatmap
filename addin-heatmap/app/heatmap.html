Heatmap

This example demonstrates how to create an add-in to visualize location or exception (rule violation) history using a heat map.

Location History
Exception History

Instructions

  1. Choose to visualize by location history or exception history.
  2. If visualizing by exception history, select a rule for which to search for exceptions.
  3. Filter the vehicle list by selecting Groups.
  4. Select the vehicle(s) to search on.
  5. Specify a date/time range.
  6. Click the Show Heat Map button to update the map.

When selecting items, use CTRL+Click to select/deselect individual items or SHIFT+Click to select a range. To select all items in the list, first select one and then use CTRL+A.
